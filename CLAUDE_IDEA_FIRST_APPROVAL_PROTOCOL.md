# Claude Workflow Decision Pitch

## Idea

Use an idea-first and approval-first workflow for all Claude coding tasks in this project:
- Claude explains the implementation idea first.
- Claude lists exact files to change.
- Claude asks for permission.
- Claude starts coding only after explicit approval.

## Why this is a strong idea

- Prevents wrong edits and scope creep.
- Protects architecture decisions and project structure.
- Improves communication and trust before changes are made.
- Reduces rework, rollbacks, and accidental file modifications.

## Recommendation

You should do this.
